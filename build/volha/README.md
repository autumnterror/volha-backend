# Приветствуем в deploy папке Volha

## Скрипты (Есть версии .cmd для Windows и .sh для Linux)
- update - скачивает обновление и перезапускает систему
- update_mig - скачивает обновление, перезапускает систему и применяет миграции
- mig_up/mig_down - применяют/откатывают миграции
- demo - заполняет бд демонстрационными данными
- restore - выполняет последний дамп
- dropdb - уничтожает бд
